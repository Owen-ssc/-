# 智慧出行地图 - 问题解决指南

## "File not found" 错误解决方案

### 方案1：直接双击打开
1. 直接双击 `index.html` 文件
2. 浏览器会自动打开地图

### 方案2：使用文件协议访问
1. 打开任意浏览器
2. 在地址栏输入：
   ```
   file:///G:/导航%20-2.0%20-%20副本%20(2)%20-%20副本/index.html
   ```

### 方案3：手动启动本地服务器
1. **确保已安装Node.js**：
   - 打开命令提示符（Win+R，输入cmd）
   - 输入 `node --version`，如果显示版本号说明已安装

2. **启动服务器**：
   - 打开命令提示符
   - 输入以下命令：
   ```
   cd /d "G:\导航 -2.0 - 副本 (2) - 副本"
   node server.js
   ```
   - 然后在浏览器访问：http://localhost:3000

### 方案4：使用Python服务器（如果已安装Python）
1. 打开命令提示符
2. 输入以下命令：
   ```
   cd /d "G:\导航 -2.0 - 副本 (2) - 副本"
   python -m http.server 8080
   ```
   - 然后在浏览器访问：http://localhost:8080

## 常见问题

### 问题：双击启动脚本显示"File not found"
**原因**：路径中包含空格和特殊字符
**解决**：使用上述方案1-4中的任意一种

### 问题：浏览器显示空白页面
**解决**：按F12打开开发者工具，查看控制台错误信息

### 问题：语音功能无法使用
**解决**：确保使用https协议或localhost访问，浏览器需要麦克风权限

## 快速启动方法

**最简单的方法**：直接双击 `index.html` 文件即可！
