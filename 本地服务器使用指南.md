# 本地服务器使用指南（空白电脑适用）

## 方案一：使用Python内置服务器（推荐）

### Windows系统
1. **检查Python是否已安装**
   - 按 `Win + R`，输入 `cmd` 回车
   - 输入 `python --version` 或 `python3 --version`
   - 如果显示版本号，说明已安装

2. **启动服务器**
   - 打开命令提示符(cmd)
   - 进入项目目录：
     ```
     cd "g:\导航 -2.0 - 副本 (2) - 副本"
     ```
   - 启动服务器：
     ```
     python -m http.server 8080
     ```
   - 或
     ```
     python3 -m http.server 8080
     ```

3. **访问地址**
   - 浏览器打开：http://localhost:8080

### 如果Python未安装
- 访问 https://www.python.org/downloads/ 下载安装
- 安装时勾选"Add Python to PATH"

## 方案二：使用Node.js内置服务器

### 1. 检查Node.js
```
node --version
npm --version
```

### 2. 启动服务器
```
cd "g:\导航 -2.0 - 副本 (2) - 副本"
npx http-server -p 8080
```

### 如果Node.js未安装
- 访问 https://nodejs.org/ 下载安装

## 方案三：使用现有server.js

项目已内置Node.js服务器，直接运行：

```
cd "g:\导航 -2.0 - 副本 (2) - 副本"
node server.js
```

访问地址：http://localhost:3000

## 方案四：使用Windows IIS（可选）

1. 启用IIS：控制面板 → 程序 → 启用或关闭Windows功能 → Internet Information Services
2. 将项目文件夹设置为IIS网站目录
3. 访问 http://localhost

## 使用步骤总结

1. **最简单**：直接双击`index.html`使用文件协议
2. **推荐**：使用Python内置服务器（无需安装额外软件）
3. **备用**：使用Node.js内置服务器

## 注意事项

- 使用本地服务器时，确保端口未被占用
- 推荐使用8080或3000端口
- 如遇到权限问题，以管理员身份运行命令提示符
- 防火墙可能会阻止访问，需要允许相应端口
