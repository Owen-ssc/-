# 智慧出行 - 老年版高德地图练习游戏

## 游戏简介

这是一个专门为老年人设计的地图使用练习游戏，模拟高德地图的界面和功能，帮助您熟悉如何使用地图应用规划出行路线。

## 游戏特点

- 🎯 **大字体设计**：所有文字都采用大字体，方便阅读
- 🎨 **简洁界面**：模仿高德地图，界面清晰易懂
- 💡 **智能提示**：如果选择太便宜的慢路线，系统会贴心提醒您
- 🗣️ **语音播报**：重要信息会有语音朗读
- 🚌 **多种出行**：支持公交、地铁、打车三种出行方式
- 🏠 **虚拟城市**：在"幸福城"中练习，不会有真实费用

## 如何使用

### 第一步：选择目的地
1. 在搜索框中输入您要去的地点
2. 可以选择：中心医院、老年活动中心、菜市场、公园、银行、超市、图书馆、社区服务中心

### 第二步：查看路线
1. 点击"搜索路线"按钮
2. 系统会为您显示3种出行方案：
   - 🚌 **公交**：最省钱，但时间较长
   - 🚇 **地铁**：快速准时，价格适中
   - 🚗 **打车**：最方便快捷，但费用较高

### 第三步：选择路线
1. 点击您想要的路线卡片
2. 查看详细信息
3. 如果有更好的选择，系统会提醒您

### 第四步：开始行程
点击"开始行程"按钮，模拟完成出行

## 温馨提示

### 省钱提示
- 如果选择公交但需要很长时间，系统会提醒您可以选择更快的地铁
- 如果选择打车费用太高，系统会提醒您有更经济的选择

### 操作技巧
- 点击地图上的📍按钮可以快速回到您的位置
- 点击地图上的+和-可以放大缩小地图
- 点击❓按钮可以查看使用帮助

### 出行建议
- **赶时间**：选择地铁或打车
- **省钱**：选择公交
- **舒适**：选择打车

## 游戏目标

通过反复练习，帮助您：
- ✅ 熟悉地图应用的基本操作
- ✅ 学会比较不同出行方式的优缺点
- ✅ 做出最适合自己的出行选择
- ✅ 增强数字时代的出行信心

如果遇到任何问题：

2. 仔细阅读操作说明
3. 多练习几次就会越来越熟练
